
 Assign Cookies (Descending Greedy Approach)

> **Problem:** LeetCode 455 - Assign Cookies  
> **Pattern:** Greedy Algorithm

# Problem Definition

You are given:

- `child[i]` → Greed factor of the *i-th child*.
- `cookies[j]` → Size of the *j-th cookie*.

A child is satisfied if

```
cookie_size >= greed_factor
```

Rules:

- Each child can receive **at most one cookie**.
- Each cookie can be used **only once**.

Return the **maximum number of satisfied children**.

---

# Algorithm

1. Sort both `child` and `cookies` in **descending order**.
2. Start with the **largest cookie**.
3. Traverse each child from the **greediest to the least greedy**.
4. If the current cookie can satisfy the child:
   - Assign the cookie.
   - Move to the next cookie.
5. Otherwise:
   - Skip the child.
6. Return the number of satisfied children.

---

# Code

```python
class Solution(object):
    def findContentChildren(self, child, cookies):
        child.sort(reverse=True)
        cookies.sort(reverse=True)

        j = 0
        res = 0

        for c in child:

            if j >= len(cookies):
                break

            if cookies[j] >= c:
                res += 1
                j += 1

        return res
```

# How It Works

```
Sort Both Arrays (Descending)
          │
          ▼
Largest Child
          │
Largest Cookie
          │
          ▼
Cookie >= Child ?
      /          \
    YES          NO
     │            │
Assign Cookie   Skip Child
     │            │
j++             Next Child
     │            │
     └──────┬─────┘
            ▼
Continue Until All Children
```

---

# Why This Works

- The arrays are sorted from **largest to smallest**.
- Always try to satisfy the **greediest child first**.
- If the **largest remaining cookie** cannot satisfy that child, then no smaller cookie can.
- Therefore, it is safe to skip that child.
- Otherwise, assign the cookie and continue.

This greedy strategy maximizes the number of satisfied children.

# Complexity Analysis

| Operation | Complexity |
|-----------|------------|

