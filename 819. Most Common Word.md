# Approach 1: Beginner-Friendly (Recommended for Learning)

### Idea

1. Convert the paragraph to lowercase.
2. Remove punctuation.
3. Split the paragraph into words.
4. Ignore banned words.
5. Count the frequency of each word using a dictionary.
6. Traverse the dictionary again to find the word with the maximum frequency.

### Algorithm

- Convert `banned` to a set.
- Convert paragraph to lowercase.
- Replace punctuation with spaces.
- Create an empty dictionary `freq`.
- For every word:
  - If the word is not banned:
    - If it already exists, increment its count.
    - Otherwise, set its count to `1`.
- Traverse the dictionary:
  - Store the highest frequency.
  - Store the corresponding word.
- Return the answer.

### Code

```python
class Solution(object):
    def mostCommonWord(self, paragraph, banned):

        banned = set(banned)

        paragraph = paragraph.lower()

        for ch in "!?',;.":
            paragraph = paragraph.replace(ch, " ")

        freq = {}

        for word in paragraph.split():

            if word not in banned:

                if word in freq:
                    freq[word] = freq[word] + 1
                else:
                    freq[word] = 1

        max_count = 0
        answer = ""

        for word in freq:

            if freq[word] > max_count:
                max_count = freq[word]
                answer = word

        return answer
```

**Time Complexity:** `O(n)`

**Space Complexity:** `O(k)`

where `k` is the number of unique words.

# Approach 2: Optimized Pythonic Solution

### Idea

This approach uses Python's built-in functions to reduce the amount of code.

Instead of writing:

```python
if word in freq:
    freq[word] += 1
else:
    freq[word] = 1
```

we use:

```python
freq[word] = freq.get(word, 0) + 1
```

Instead of manually finding the maximum frequency:

```python
max_count = 0
answer = ""
...
```

we use:

```python
max(freq, key=freq.get)
```

which returns the key having the largest value.

### Algorithm

- Convert banned words into a set.
- Convert paragraph to lowercase.
- Remove punctuation.
- Split into words.
- Count frequencies using `dict.get()`.
- Return the word with the highest frequency using `max()`.

### Code

```python
class Solution(object):
    def mostCommonWord(self, paragraph, banned):

        banned = set(banned)

        paragraph = paragraph.lower()

        for ch in "!?',;.":
            paragraph = paragraph.replace(ch, " ")

        freq = {}

        for word in paragraph.split():

            if word not in banned:
                freq[word] = freq.get(word, 0) + 1

        return max(freq, key=freq.get)
```

**Time Complexity:** `O(n)`

**Space Complexity:** `O(k)`
