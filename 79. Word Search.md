# 79. Word Search

**Difficulty:** Medium

---

## Problem Statement

Given an `m x n` grid of characters `board` and a string `word`, return `true` if the word exists in the grid.

The word can be constructed from letters of sequentially adjacent cells, where adjacent cells are **horizontally or vertically** neighboring.

The **same cell cannot be used more than once**.

---

## Example

### Example 1

```text
Input:
board =
[
 ["A","B","C","E"],
 ["S","F","C","S"],
 ["A","D","E","E"]
]

Output:
true
```

# Algorithm

1. Find the number of rows and columns.
2. Store the four possible directions:
   - Up
   - Down
   - Left
   - Right
3. Start DFS from every cell.
4. Base Case:
   - If `index == len(word)`, return `True`.
5. Return `False` if:
   - Out of bounds.
   - Current character does not match.
6. Store the current character.
7. Mark the current cell as visited (`#`).
8. Visit all four neighboring cells recursively.
9. Restore the original character (Backtracking).
10. If any path matches the complete word, return `True`.
11. Otherwise return `False`.

---

# Python Code

```python
class Solution(object):
    def exist(self, board, word):

        rows = len(board)
        cols = len(board[0])

        directions = [
            (-1, 0),   # Up
            (1, 0),    # Down
            (0, -1),   # Left
            (0, 1)     # Right
        ]

        def dfs(r, c, index):

            if index == len(word):
                return True

            if (
                r < 0 or
                r >= rows or
                c < 0 or
                c >= cols or
                board[r][c] != word[index]
            ):
                return False

            temp = board[r][c]
            board[r][c] = "#"

            found = False

            for dr, dc in directions:

                nr = r + dr
                nc = c + dc

                if dfs(nr, nc, index + 1):
                    found = True
                    break

            board[r][c] = temp

            return found

        for r in range(rows):
            for c in range(cols):
                if dfs(r, c, 0):
                    return True

        return False
```

---

# Time Complexity

```text
O(M × N × 4^L)
```

- `M` = Number of rows
- `N` = Number of columns
- `L` = Length of the word

---

# Space Complexity

```text
O(L)
```

- Recursive call stack depth equals the length of the word.

---

# Key Points

- DFS + Backtracking
- Explore 4 directions
- Mark visited cells using `#`
- Restore the cell after recursion
- Cannot reuse the same cell
- Start DFS from every cell
