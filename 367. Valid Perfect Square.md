## Python Solution

```python
class Solution(object):
    def isPerfectSquare(self, num):
        low = 1
        high = num

        while low <= high:
            mid = (low + high) // 2
            square = mid * mid

            if square == num:
                return True
            elif square < num:
                low = mid + 1
            else:
                high = mid - 1

        return False
```

## Time Complexity

- **Time:** O(log n)
- **Space:** O(1)
